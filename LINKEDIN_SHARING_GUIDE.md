# LinkedIn Sharing Guide

## Overview
This guide will help you share your Tic Tac Toe with a Twist game on LinkedIn to showcase your development skills.

## Step 1: Deploy Your Game First

Before sharing on LinkedIn, make sure your game is live and accessible:

### Option A: Deploy to Vercel (Recommended)

1. **Push to GitHub** (if not already done):
   ```bash
   git add .
   git commit -m "Ready for deployment"
   git push origin main
   ```

2. **Deploy via Vercel Dashboard:**
   - Go to https://vercel.com/dashboard
   - Click "Add New Project"
   - Import your GitHub repository: `raghavdharani/TTT`
   - Vercel will auto-detect and deploy
   - Your game will be live at: `https://ttt-xxx.vercel.app`

3. **Custom Domain (Optional):**
   - Add a custom domain in Vercel settings
   - Example: `tictactoe-twist.yourname.com`

### Option B: Other Hosting Options

- **Netlify**: Similar to Vercel, free hosting
- **GitHub Pages**: Free but requires some setup
- **Your own domain**: If you have web hosting

## Step 3: Add to Your LinkedIn Profile

### As a Featured Item

1. Go to your LinkedIn profile
2. Scroll to "Featured" section
3. Click "+" to add
4. Select "Link"
5. Add:
   - **Title**: "Tic Tac Toe with a Twist"
   - **URL**: Your deployed game URL
   - **Description**: Brief description of the project

### As a Project

1. Go to your profile
2. Scroll to "Projects" section (or add it)
3. Click "+" to add project
4. Fill in:
   - **Project Name**: Tic Tac Toe with a Twist
   - **Description**: 
     ```
     A strategic twist on the classic Tic Tac Toe game built with React.
     Features include token limits, movement restrictions, and series gameplay.
     ```
   - **URL**: Your deployed game URL
   - **Start Date**: When you started
   - **End Date**: When you completed it
   - **Associated with**: (Optional - your company/school)

### In Your Experience Section

If this was part of a job or learning program:

1. Edit the relevant experience entry
2. Add it as an accomplishment or project
3. Link to the game
