# ShiftTacToe - AI Design Generator Prompt (Concise)

## Direct Prompt for AI Design Tools

```
Design a comprehensive, modern UI for "ShiftTacToe" - a strategic Tic-Tac-Toe variant game. 

STYLE: Simplistic yet mystical, modern elegance, smooth transitions, ethereal aesthetic, minimalist with purposeful details, premium polished feel, enchanting but not overwhelming.

COLOR PALETTE:
- Background: Deep navy with subtle purple undertones OR warm off-white with blue tint (mystical quality)
- Surface/Cards: Elevated glassmorphism with subtle transparency and soft shadows
- Primary Actions: Soft blue/purple with mystical glow
- X Token: Solid red (#dc2626) with subtle pulse animation
- O Token: Solid blue (#2563eb) with slow rotation animation
- Accents: Gentle greens (success), warm amber (attention), soft reds (errors)
- Text: High contrast for readability (near-black or pure white)

TYPOGRAPHY: Modern sans-serif (Inter/Poppins style), clean and readable, mystical character, clear hierarchy (48-64px titles, 16px body, 14px secondary).

COMPONENTS TO DESIGN:

1. HOME/SETUP SCREEN:
   - Centered layout with "ShiftTacToe" title (mystical styling)
   - 3 play mode buttons (2 Player, VS Computer, Online) - glassmorphism cards
   - Progressive disclosure: Difficulty selector (Easy/Hard/Insane), Game mode (Single/Best of 3/Best of 5), Starting player (X/O tokens)
   - Start button with mystical glow
   - Smooth transitions between selections
   - Selected states: Enhanced glow, gentle pulse, mystical aura

2. GAME BOARD:
   - 3x3 grid with rounded corners, subtle borders/shadows
   - Squares: Clean, modern, hover effects (scale 1.05, glow increase)
   - Token placement: Scale 0→1 with bounce, fade-in with glow (300-400ms)
   - Token movement: Smooth slide with slight arc, glow trail (400-500ms)
   - Visual states: Enabled (subtle pulse/glow), Disabled (muted), Relocating (highlighted), Valid destinations (subtle highlight)
   - Board feels like sacred game space

3. GAME STATUS:
   - Prominent but elegant display
   - Current player indicator with animated tokens
   - Status messages (turn, win, draw) with smooth transitions
   - Series score display (Best of 3/5) - elegant, clear
   - Win celebration: Enhanced glow, pulse, subtle particles

4. ONLINE SETUP:
   - Create/Join room interface
   - Connection status with mystical indicators
   - Room ID display (copyable, styled)
   - Waiting animations (subtle pulse, shimmer)
   - Consistent with main setup

5. BUTTONS & ACTIONS:
   - Consistent styling throughout
   - Hover: Scale 1.02-1.05, glow increase, color shift (200ms)
   - Active: Scale 0.98, brief feedback
   - Primary actions: Mystical glow, smooth transitions
   - Touch-friendly (44x44px minimum)

ANIMATIONS:
- Duration: 200-400ms (smooth, not sluggish)
- Easing: Elegant curves (ease-in-out, custom cubic-bezier)
- Token X: Continuous subtle pulse
- Token O: Slow rotation (3s per rotation)
- Screen transitions: Fade 300-400ms, optional subtle slide/scale
- All interactions: Smooth, purposeful, delightful

VISUAL EFFECTS:
- Glassmorphism: Frosted glass cards with backdrop blur
- Shadows: Soft, layered for depth, mystical glow shadows
- Gradients: Subtle background gradients (if used)
- Borders: Rounded corners throughout, subtle or no borders
- Glow: Colored soft glows for selected/active states

RESPONSIVE:
- Mobile-first: Touch-friendly, stacked layouts, readable sizes
- Tablet: Balanced layout, optimal spacing
- Desktop: Generous spacing, hover states, keyboard support
- Future mobile app ready: Translates well to native patterns

ACCESSIBILITY:
- High contrast (WCAG AA)
- Clear focus indicators
- Keyboard navigation
- Respects prefers-reduced-motion
- Color not only indicator

DELIVERABLES NEEDED:
- Visual mockups (mobile, tablet, desktop)
- Component specifications (colors, typography, spacing, animations)
- Interaction states (hover, active, disabled, selected)
- Animation guidelines (duration, easing, keyframes)
- Design system (color palette, typography scale, spacing, components)
- Style guide document

MOOD: Enchanting digital artifact, mystical game experience, premium feel, timeless yet modern, joyful interactions, beautiful craftsmanship.
```
